# 🌐 Bio Link 
Proyek ini merupakan halaman **bio link sederhana** menggunakan HTML, CSS, dan JavaScript. Card menampilkan foto, nama, profesi, serta tombol media sosial yang dapat diklik.

---

## ✨ Fitur

- Foto profil berbentuk lingkaran
- Nama dan deskripsi profesi
- Tombol media sosial (Instagram, LinkedIn, YouTube, TikTok)
- Tombol portofolio
- Hover animation pada tombol
- Responsive design

---

## 🛠️ Teknologi yang Digunakan

- HTML5
- CSS3 (Flexbox & CSS Variables)
- JavaScript (Event Listener)
- Font Awesome (ikon)

---

## 📂 Struktur Folder

```
project-folder/
│
├── index.html
├── style.css
├── script.js
├── gambar.jpg
└── README.md
```

---


---

## 🚀 Cara Menjalankan

1. Download atau clone repository ini.
2. Pastikan semua file berada dalam satu folder.
3. Buka file `index.html` di browser.
4. Klik tombol media sosial untuk membuka link di tab baru.

---

## 🎯 Preview Fitur Interaktif

- 🖱️ Hover pada tombol → warna berubah & tombol terangkat
- 🔘 Klik tombol → membuka website sesuai platform

---

## 📌 Catatan

- Pastikan koneksi internet aktif untuk memuat ikon dari CDN Font Awesome.
- Anda bisa mengganti link media sosial pada file `script.js`.
